Краткое описание по проекту:

TankBasic - базовый класс для игрока и для противников;
Классы менеджеров и Pool реализуют паттерн Singleton для удобства доступа;
Для повторного использования ресурсов используется Pool;
При попадании пули, в последний кадр анимации встроено обращение к скрипту, который
помещает пулю в пул.
